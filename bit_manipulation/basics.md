
## Convert to Binary:

$(13)_{10}$ -> $(1101)_2$

$(4)_{10}$ -> $(100)_2$

Conversion:
num = 13

num % 2 = 1 & num = num // 2 = 6

num % 2 = 0 & num = num // 2 = 3

num % 2 = 1 & num = num // 2 = 1

num % 2 = 1 & num = num // 2 = 0

remainders from bottom-up: 1101

Conversion: num = 4

num % 2 = 0 & num = num // 2 = 2

num % 2 = 0 & num = num // 2 = 1

num % 2 = 1 % num = num // 2 = 0

remainders from bottom up: 100

## Convert to Base-10

1101 = $1 \times 2^{3}$ + $1 \times 2^{2}$ + $0 \times 2^{1}$ + $1 \times 2^{0}$ 

1101 = 13
